# Usage
<!-- usage -->
```sh-session
$ npm install -g qrl-cli
$ qrl-cli COMMAND
running command...
$ qrl-cli (-v|--version|version)
qrl-cli/0.0.0 darwin-x64 node-v8.11.3
$ qrl-cli --help [COMMAND]
USAGE
  $ qrl-cli COMMAND
...
```
